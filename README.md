# Drift Risk Assessment UI

This project is a sandbox to explore Signal/Slot communication within a QWidget.

## Model

### Input Fields

QComboBox for each input:

- Temperature
- Wind Speed
- Humidity
- Tractor Speed
- Pressure Altitude
- Nozzle

### Summary Display

Indicates the combined risk score of the input fields.

### Score Breakdown

- Low Risk: 0 - 2
- Exercise Caution: 3 - 6
- Dangerous Conditions: 6+
